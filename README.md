# RESUME_GENERATOR 📄

A dynamic, web-based Resume/CV generator built with PHP. This application allows users to input their personal, educational, and professional details through an HTML form, upload a profile picture, and instantly generate a beautifully styled resume.

## 🚀 Features

* **Dynamic Data Handling:** Utilizes PHP `$_POST` to capture and process text data from the HTML form.
* **Image Uploads:** Securely handles profile picture uploads using the PHP `$_FILES` superglobal.
* **Responsive Design:** Styled with custom CSS to ensure the generated CV looks professional and is easy to read.
* **Instant Generation:** Processes the form and renders the CV on the fly.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3
* **Backend:** PHP (8.x)
* **Environment:** XAMPP / Localhost (for local development)
